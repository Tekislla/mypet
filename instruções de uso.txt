- instalar o java (https://drive.google.com/file/d/1HyeOaCAFK56bi6N9U-7VLQ02iVzCGO0M/view?usp=sharing)
- baixar o app (https://drive.google.com/file/d/1lziucsb8NWORnEnsbjBEXZLXeojeuX4r/view?usp=sharing)
- baixar o repositório (https://github.com/Tekislla/mypet)
- instalar o postman (https://drive.google.com/file/d/1u7TkUC6kkzdBzsLpfxvecGwJcI5px-xc/view?usp=sharing)
- instalar pgadmin (https://drive.google.com/file/d/1pJlmL6_UP4pAcNGpi8fNqBPtgQLzeaCy/view?usp=sharing)
- instalar o docker (https://drive.google.com/file/d/1taFyKDGeMZ1Ftu386oqdZ0iDxcgzUvDu/view?usp=sharing)
	- abrir o cmd e executar: docker run -p 5432:5432 --name postgres -e POSTGRES_PASSWORD=postgres -d postgres

- abrir o cmd e ir até a pasta onde está o app
	- digitar o comando: java -jar mypet-0.0.1-SNAPSHOT.jar

- abrir os arquivos html na pasta FRONTEND do repositório (apenas signup e addpet)
	- cadastrar o usuário
	- cadastrar o pet

- ir até o postman
- digitar localhost:5000/mypet/person/find-all ----- (encontrar todos os usuários)
	- localhost:5000/mypet/person/find-by-id/{id a ser passado} ----- (encontrar usuário por id)
	- localhost:5000/mypet/person/delete-person/{id a ser passado} ----- (deletar usuário por id)


	- localhost:5000/mypet/pet/find-all ----- (encontrar todos os pets)
	- localhost:5000/mypet/pet/find-by-id/{id a ser passado} ----- (encontrar pet por id)
	- localhost:5000/mypet/person/delete-pet/{id a ser passado} ----- (deletar pet por id)
